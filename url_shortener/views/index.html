API Basejump: URL Shortener Microservice

User stories:
    1) I can pass a URL as a parameter and I will recieve a shortened URL in the JSON response.
    2) If I pass a invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3) When I visit the shortened URL, it will redirect me to my original link.

Example usage:

my.app/https://www.google.com

Example output:

{ "Orignal URL": https://www.google.com, "Short URL": "https://www.shorturl.sddlkf.com" }
